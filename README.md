# Внешняя сортировка большого файла (2 ГБ)

Этот проект реализует **внешнюю сортировку** большого файла (например, 2 ГБ)  
без возможности загружать его полностью в оперативную память.

Сортировка выполняется **текстовых или бинарных данных** с использованием:

- разбиения на отсортированные куски,
- сортировки каждого куска в памяти,
- попарного слияния файлов (external merge sort).

---

## Особенности реализации

- Генерация большого файла размером ~2 ГБ.
- Ограничение: в память помещается только небольшой блок данных.
- Сортировка производится в несколько этапов:
  - Разбиение файла на части фиксированного размера.
  - Сортировка каждой части (`std::sort`).
  - Попарное слияние всех файлов до получения одного результата.
- Использованы указанные в задании подсказки:
  - `ofstream::trunc`
  - чтение файла через `while (!fs.eof())`
  - удаление временных файлов через `std::remove`

## Как запустить

1. Открыть решение `SortBigFile.sln` в Visual Studio.
2. Собрать проект в Debug или Release.
3. Запустить `main.cpp`.
